Java VS Testes
Sua tarefa é implementar a classe Ai. 
Esta classe não deve usar o construtor padrão. 
Ai deve ter duas propriedades, ambas Strings: creationDate e name. 
Seu construtor personalizado deve definir valores para essas propriedades. 
A classe também precisa ter os getters e setters necessários para ambas as variáveis ​​de instância.


- Propriedades: A classe Ai deve ter duas propriedades do tipo String: creationDate e name. Elas serão inicializadas através de um construtor personalizado.👍
- Getters e Setters: Criar métodos de acesso e modificação para as duas propriedades.👍
- Métodos solicitados:

    doubleArray: Duplicar os valores de um array de inteiros.👍
    factorialIterative: Calcular o fatorial de forma iterativa.👍
    factorialRecursive: Calcular o fatorial de forma recursiva.👍
    palindromeReplace: Substituir palavras que são palíndromos em uma string.
    emailValidator: Retornar uma expressão regular (regex) que valida e-mails.
